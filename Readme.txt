USAGE:
1. Run Skype 2.2.0.35
2. Wait login window
3. Run skype2-mempatch `pidof skype`
4. Enter username and password, check 'Sign me in when Skype starts'
5. After Skype signing in, close skype or wait of crash :)
6. Restart Skype
